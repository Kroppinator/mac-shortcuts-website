Mac Quick Reference

Terminal commands and keyboard shortcuts for Mac newcomers

Navigation & Files

pwd

Print working directory - shows your current folder path

cd [path]

Change directory - navigate to a different folder

ls

List files and folders in the current directory

ls -la

List all files including hidden ones with details

mkdir [name]

Create a new directory (folder)

cd ..

Go up one directory level to the parent folder

File Management

cp [source] [destination]

Copy a file or directory

mv [source] [destination]

Move or rename a file or directory

rm [file]

Delete a file (be careful - no recovery!)

rm -r [directory]

Delete a directory and all its contents

touch [filename]

Create an empty file

cat [file]

Display the contents of a file

Text & Search

grep [text] [file]

Search for text within a file

find [path] -name [name]

Search for files by name in a directory

sed -i '' 's/[old]/[new]/g' [file]

Replace text in a file

nano [file]

Open a simple text editor to edit a file

wc -l [file]

Count the number of lines in a file

sort [file]

Sort lines in a file alphabetically

System Information

whoami

Display the current user account name

date

Show the current date and time

df -h

Show disk space usage in human-readable format

du -sh [path]

Calculate the size of a directory or file

top

Display running processes and system usage

uname -a

Show system information (OS type, kernel version)

Permissions & Ownership

chmod [permissions] [file]

Change file permissions (who can read/write/execute)

chown [user] [file]

Change file ownership

sudo [command]

Run a command with administrator privileges

ls -l [file]

View file permissions and ownership details

chmod +x [file]

Make a file executable

chmod -x [file]

Remove execute permission from a file

Window & App Management

⌘Tab

Switch between open applications

⌘~

Switch between windows of the same app

⌘M

Minimize the current window

⌘W

Close the current window

⌘Q

Quit the current application

⌘H

Hide the current application

Text Editing

⌘Z

Undo the last action

⌘ShiftZ

Redo the last action

⌘C

Copy selected text or item

⌘X

Cut selected text or item

⌘V

Paste from clipboard

⌘A

Select all content

Search & Find

⌘F

Open find dialog to search in document

⌘G

Find the next occurrence of the search term

⌘ShiftG

Find the previous occurrence of the search term

Space⌘

Open Spotlight search to find files and apps

⌘,

Open preferences or settings

⌘OptionEsc

Force quit an application

File & Finder

⌘N

Open a new window or document

⌘O

Open a file

⌘S

Save the current document

⌘ShiftS

Save as (choose location and name)

⌘Delete

Move item to Trash (in Finder)

⌘OptionDelete

Empty Trash (permanently delete items)

Browser Navigation

⌘T

Open a new browser tab

⌘N

Open a new browser window

⌘W

Close current tab

⌘ShiftW

Close current window

⌘Option→

Go to next open tab

⌘Option←

Go to previous open tab

💡 Tip: Copy any terminal command to use in your Terminal app. Learn more at Apple Support
